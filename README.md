# Praxis - przewodniki praktyczne MateMatic

**Praxis** to otwarta linia przewodnikow praktycznych dla polskich kancelarii prawnych,
wydawana przez [MateMatic](https://matematicsolutions.com). Wiedza o bezpiecznym uzyciu AI
w kancelarii nie powinna byc produktem. Powinna byc dobrem wspolnym - i to repozytorium
jest jego domem.

## Dwa formaty

- **Mapy** - zywe, przeszukiwalne zestawienia narzedzi i zasobow (np. mapa polskiego LegalTech AI).
- **Podreczniki** - proceduralne przewodniki krok po kroku, z checklistami i artefaktami do pobrania.

## Trzy zasady

1. **Aktywne, nie statyczne** - to narzedzia pracy, nie zamrozone PDF-y.
2. **Subiektywne, nie obiektywne** - z jawnym komentarzem i ogladem MateMatic.
3. **Zywe, nie zamrozone** - wersjonowane, rozwijane z uwagami czytelnikow.
